# Pizza Sales Database Project

## Description

This project contains a MySQL database for pizza sales analysis.
The database includes tables for orders, order details, pizzas, and pizza types.

## Tools Used

* MySQL Workbench
* MySQL Server

## Files Included

* sql pizza ales project.sql

## How to Use

1. Open MySQL Workbench
2. Open the .sql file
3. Execute the script to create the database
